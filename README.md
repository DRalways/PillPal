# PillPal
 PillPal – Medicine Tracker

A desktop application designed to help users efficiently manage and track their daily medications.
PillPal simplifies medication routines by allowing users to organize medicines, monitor schedules, and maintain a history of intake.

---

##  Features

* Add and manage medicines
* Set and track medication schedules
* Maintain medication history
* Simple dashboard for quick overview
* Lightweight and easy-to-use interface

---

##  Tech Stack

* **Java**
* **JavaFX** (User Interface)
* **File-based Storage System**

---

## 📁 Project Structure

```
src/medicinetracker   → Core application logic  
data/                 → Stores medicine data  
resources/            → UI assets and configurations  
```

---

##  Purpose

This project was developed to explore building practical desktop applications focused on healthcare.
It aims to improve daily medication management and provide a structured way for users to maintain their health routines.
